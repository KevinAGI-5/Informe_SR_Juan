# Informe_SR_Juan

📊 Análisis de Rendimiento de Tiendas - Caso "Sr. Juan"
Este repositorio contiene un análisis exhaustivo de datos utilizando Python y Pandas para evaluar el rendimiento de cuatro tiendas comerciales. El objetivo principal es asesorar al Sr. Juan sobre cuál es la mejor plataforma para comercializar sus productos basándose en métricas de ventas, satisfacción del cliente y costos logísticos.

🎯 Objetivo del Proyecto
Determinar la viabilidad comercial y el retorno de inversión potencial mediante el análisis de:

Ingresos Totales: Volumen de facturación por sucursal.

Penetración de Categorías: Identificación de nichos dominantes (Muebles, Electrónicos, etc.).

Customer Experience: Análisis de calificaciones promedio (Rating).

Eficiencia Logística: Comparativa de costos de envío promedio.

🛠️ Tecnologías Utilizadas
Python 3.x

Pandas: Para la manipulación y limpieza de estructuras de datos.

Matplotlib: Para la generación de visualizaciones (Gráficos de barras, líneas y circulares).

Google Colab: Entorno de desarrollo.

📈 Hallazgos Clave
1. Liderazgo en Ventas
La Tienda 1 lidera la facturación total con $1,150,880,400, seguida de cerca por la Tienda 2. No obstante, el volumen de ingresos no es el único factor de decisión.

2. Preferencias del Consumidor
Se identificó que las categorías de Muebles (1,886 ventas) y Electrónicos (1,772 ventas) son los motores principales de demanda en todas las ubicaciones.

3. Satisfacción y Logística
Tienda 3 destaca con la mejor calificación promedio (4.05), sugiriendo una mayor fidelización.

Tienda 4 presenta la operación logística más económica con un costo de envío promedio de $23,459.46.

💡 Recomendación Final
El análisis concluye que la Tienda 3 es la opción estratégica más sólida. A pesar de no tener el ingreso más alto, su liderazgo en la categoría de Muebles y su superioridad en satisfacción del cliente ofrecen un entorno de menor riesgo y mayor crecimiento a largo plazo para los productos del Sr. Juan.
